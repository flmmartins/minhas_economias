# How to run

```
pipenv install
```

You might need to change the env variable in .env file to scan the directory with files. Running the command below should load .env file:

```
pipenv shell
```

Run

```
python3 convert.py
```

# How to upgrade python

Change Pipfile to version


```
 brew upgrade pipenv
 pipenv --python 3.11
 pip install pipenv
 pipenv update
 pipenv upgrade

```
